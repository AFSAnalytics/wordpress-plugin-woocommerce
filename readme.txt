== Installation ==

1. Install [AFS Analytics](https://www.afsanalytics.com/ "AFS Analytics") plugin either via the WordPress plugin directory, or by uploading the files to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. From your Wordpress admin dashboard, choose to create a free account or, if you already have one, to link it.
4. Access to your website statistics through the **AFS Analytics** link on WordPress menu.
